## Opplastningsmoduler og elementer

Det sirkulære opplastningselementet skal brukes på små flater og som ikon.

    <div class="white--loader loader--white loader--size loader--pos--white"></div>

## Filopplastningselementer

### Linjært opplastningselement uten verdi
Linjært opplastningselementet bruker klassen "progressbar".

<div class="progressbar__container">
    <div class="progressbar"></div>
</div>
<br/>

```markdown
<div class="progressbar__container">
    <div class="progressbar"></div>
</div>
```

### Linjært opplastningselement med verdi
Opplastningselementet bruker ::after selector for å sette inn verdi og fyll farge.
I dette eksemplet er "demo" klassen inneholder: .demo::after {
width: 50%;
background: var(--blaa_moerk);
}

<div class="progressbar__container">
    <div class="progressbar demo"></div>
</div>
<br/>

```markdown
<div class="progressbar__container">
    <div class="progressbar demo"></div>
</div>

<style>
.demo::after {
    width: 50%;
    background: var(--blaa_moerk);
}
</style>
```

## Listevisning

### Stor skjerm
Trekk ut en ferdig liste fra Assets, og endre innholdet og tilpass størrelsen til ditt design.
Scroll: Liste skal vise maks 12 rader, etter 12 rader skal scroll brukes (se under)
Horisontal scroll skal aldri brukes!
Celle med tall: Høyrestilt innhold
Celle med tekst: Venstrestilt innhold
Celle med ikoner:  Ikoner skal alltid værepå venstre side av teksten
Blank celle: Brukes der det er behov for celler uten innhold (se “slett” under)
Celle med avsjekksliste: Avsjekksliste skal ha en Slett alle funksjon (se under)

En liste skal aldri scrolles horisontalt, eller gå over designets maks grid bredde.
Selvom designet i Figma ikke viser sticky header, skal dette kodes inn i ferdig design.
