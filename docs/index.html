Kartverkets interaksjons og designbibliotek

På denne siden kommer det informasjon om Kartverkets interaksjons og designbibliotek. Denne informasjonen vil omfatte guidelines for design i Kartverket, inkludert CSS og SCSS, og gjennbrukbare ressurser og komponenter. Komponentene er laget med angular og kan brukes som de er, men vi vil også beskrive oppbyggningen til hvert enkelt komponent, slik at det skal være mulig å implementere de samme komponentene også i produkter som ikke bruker Angular.
