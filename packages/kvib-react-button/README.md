# Button (React)

## Innstallering

```bash
$ npm install @kartverket/kvib-react-button
```

## Bruk

```tsx
import { Button } from "@kartverket/kvib-react-button";
```


### `<Button />` props

#### `variant`
- `primary` (default)
- `secondary`
- `tertiary`

#### `colorScheme`
- `green` (default)
- `blue`

#### `size`
- `sm` (default)
- `md`
