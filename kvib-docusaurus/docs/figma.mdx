## Om KVIB Figma

KVIB står for Kartverkets Interaksjonsbibliotek. Biblioteket består av alle komponenter og elementer som er designet for Kartverket. Spesialdesignede elementer ligger ute i prosjektene, finnes i katalogen.

#### Om størrelser

**Extra small**
* Dashboard
* Apper
* Kartløsninger
* Sidemenyer

**Small**
* Mobil

**Medium**
* Tablets

**Large**
* Desktop
