Installasjon av KVIB-CSS GitHub Package​

GitHub​

  1. Opprett et Personal Access Token (PAT). Se Github dokumentasjon for å opprette PAT
  2. Autoriser PAT for bruk med organisasjonen. Se Github dokumentasjon for PAT autentisering

npm​

  1. Opprett en .npmrc fil i prosjektmappen og legg til følgende:
@kartverket:registry=https://npm.pkg.github.com
  1. I prosjektmappen som inneholder .npmrc filen, logg på npm med GitHub brukernavn og PAT ved å bruke følgende kommando:
npm login --scope=@kartverket --registry=https://npm.pkg.github.com
  1. Kjør npm install. Biblioteket er nå klart til bruk.
npm install
  1. Importer eller legg til link til biblioteket i CSS eller HTML filen i prosjektet.
  • I HTML fil:
<link rel="stylesheet" href="PATH/node_modules/@kartverket/kvib/main.css">
  • I CSS fil:
@import 'PATH/node_modules/@kartverket/kvib/main.css';
