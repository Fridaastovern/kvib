Installasjon av KVIB-CSS Github Package​

Github​

  1. Opprett et Personal Access Token (PAT). Se Github dokumentasjon for å opprette PAT
  2. Autoriser PAT for bruk med organisasjonen. Se Github dokumentasjon for PAT autentisering

npm​

  1. Opprett et .npmrc fil i prosjekt mappen og legg til følgende:
@kartverket:registry=https://npm.pkg.github.com
  1. I prosjekt mappen som inneholder .npmrc filen, logg på npm med Github brukernavn og PAT ved å bruke følgende kommando:
npm login --scope=@kartverket --registry=https://npm.pkg.github.com
  1. Skriv npm install og da skal biblioteket være klar til bruk.
npm install
